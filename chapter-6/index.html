6.1 Introduction to Linear Filters

Linear filters process images by applying mathematical operations to a local neighborhood of pixels using a mask (or kernel).

Example with 5x5 Matrix


>> x = uint8(10 * magic(5))    % Create test matrix
>> mean2(x(1:3,1:3))           % Calculate average of first 3x3 region
>> mean2(x(1:3,2:4))           % Calculate average of shifted region

6.2 Filter Notation

Linear filters are described using coefficient matrices that specify how pixel values within the mask are combined.

Averaging Filter Example


[ 1/9  1/9  1/9 ]
[ 1/9  1/9  1/9 ]
[ 1/9  1/9  1/9 ]

Each coefficient represents the weight applied to the corresponding pixel in the mask.

6.3 Handling Image Edges

Special consideration is needed when applying filters near image edges where the mask extends beyond the image boundaries.

Common Solutions

  • Padding with zeros
  • Extending edge values
  • Mirroring the image
  • Circular wrapping

Considerations

  • Impact on filter output
  • Edge preservation
  • Computational efficiency
  • Application requirements

6.4 Filter Implementation

MATLAB Implementation


% Define a 3x3 averaging filter
h = ones(3,3) / 9;

% Apply filter to image
filtered_img = imfilter(original_img, h);

% Apply filter with edge handling
filtered_img = imfilter(original_img, h, 'replicate');

Key Functions

  • imfilter: Apply linear filter to image
  • conv2: 2D convolution
  • fspecial: Create predefined filters
  • medfilt2: 2D median filtering
