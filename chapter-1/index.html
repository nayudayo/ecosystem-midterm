Chapter 1: Introduction and Fundamentals

Introduction to Digital Image Processing

Digital Image Processing is the manipulation of digital images through computer algorithms. It focuses on enhancing image quality, extracting useful information, and facilitating image analysis.

In digital image processing, we focus on single images or 'snapshots.' These images can represent anything—people, nature, microscopic components, or medical scans. Every image contains valuable information that can be analyzed and understood.

Visual Perception

Humans rely heavily on vision to understand the world. We can quickly identify faces, distinguish colors, and process large amounts of visual information. However, what we see is constantly changing—buildings, landscapes, and even shadows appear different depending on the time of day or lighting.

Key Applications

  • Medical imaging
  • Remote sensing
  • Digital photography

Purpose

  • To enhance image quality
  • Extract useful information
  • Facilitate image analysis

What is Image Processing?

Image processing involves changing the nature of an image to either:

  1. Improve its pictorial information for human interpretation
  2. Render it more suitable for autonomous machine perception

Key Aspects

  • Uses computer algorithms to process digital images
  • Involves two distinct but equally important aspects
  • Different requirements for humans vs machines

Human vs Machine Preferences

  • Humans prefer sharp, clear, and detailed images
  • Machines work better with simple, uncluttered images
  • A procedure that makes an image 'look better' may be unsuitable for machine processing

Types of Image Processing

1. Enhancing Image Sharpness

Image sharpening enhances the edges of an image to make it appear clearer and more defined. This is particularly important in printing where images need to appear at their best on the printed page.

Left: Original image, Right: After sharpening

2. Removing Noise

Noise refers to random errors in the image that can occur during data transmission. Different types of noise require different removal methods.

Left: Image with noise, Right: After noise removal

3. Removing Motion Blur

Motion blur occurs when the shutter speed is too slow for the moving object. Deblurring techniques can help recover details like numberplates and other fine features.

Left: Motion blurred image, Right: After deblurring

4. Edge Detection

Edge detection is crucial for measuring objects in images. It helps identify boundaries and calculate areas within them.

Types of Digital Images

Binary Images

Each pixel is either black or white, requiring only one bit per pixel. Ideal for:

  • Text documents
  • Fingerprints
  • Architectural plans

Example of a binary image showing edges in white against a black background

Greyscale Images

Each pixel represents a shade of grey from 0 (black) to 255 (white). Common in:

  • Medical X-rays
  • Printed works
  • Natural object recognition

Example of a greyscale image showing an alleyway scene

RGB Color Images

Each pixel has red, green, and blue components (24-bit color), allowing for 16,777,216 possible colors.

Example of an RGB color image showing rich color variety

Indexed Color Images

Uses a color map or palette, with each pixel storing an index to the color rather than the color itself.

Example of an indexed color image with its corresponding color indices

Image File Sizes

Image files tend to be large. Here's how different image types affect file size:

Binary Image (512 x 512)

  • Total bits: 512 x 512 x 1 = 262,144
  • Size in bytes: 32,768 bytes
  • Size in KB: 32.768 KB
  • Size in MB: 0.033 MB

Greyscale Image (512 x 512)

  • Total bits: 512 x 512 x 8
  • Size in KB: 262.14 KB
  • Size in MB: 0.262 MB

Color Image (512 x 512)

  • Total bytes: 512 x 512 x 3 = 786,432
  • Size in KB: 786.43 KB
  • Size in MB: 0.786 MB

Information Equivalence

A picture worth 1000 words equals:

  • 1000 words × 10 chars/word
  • 10,000 chars × 8 bits/char
  • Total: 80,000 bits of information

Image Acquisition

CCD Camera

Uses an array of photosites that generate voltage proportional to light intensity. Capable of capturing images in up to one ten-thousandth of a second.

Flatbed Scanner

Uses a single row of photosites moved across the image, capturing it row-by-row. Slower but more controlled process.

Image Perception

Image perception involves two basic steps:

  1. Capturing the image with the eye
  2. Recognizing and interpreting the image with the visual cortex in the brain

Background Effect

A single block of grey will appear:

  • Darker on white background
  • Lighter on black background
  • We perceive grey scales relative to their surroundings

Continuous Gradients

In continuously varying grey levels:

  • We may observe non-existent intensities
  • Our eyes perceive horizontal edges even in smooth gradients
  • Light to dark transitions create visual artifacts

Boundary Effects

At intensity boundaries:

  • Visual system tends to overshoot or undershoot
  • Creates perceived intensity differences
  • Affects interpretation of object boundaries
