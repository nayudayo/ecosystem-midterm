Basic Use of MATLAB

Chapter 2: MATLAB Fundamentals

2.1 Introduction

MATLAB is a data analysis and visualization tool which has been designed with powerful support for matrices and matrix operations. It has excellent graphics capabilities, and its own powerful programming language.

Example Commands


>> sin(pi/3)              % Calculates the sine of pi/3
>> c = imread('cameraman.tif')    % Reads an image file
>> a = imclose(b)         % Applies morphological closing

Key Concepts

  • MATLAB uses toolboxes - sets of programs designed to support particular tasks
  • Functions are keywords that accept parameters and produce output
  • Commands are specific uses of functions
  • All data in MATLAB is treated as matrices

Data Types in MATLAB

  • Images: Matrices of grey values or RGB values
  • Single values: Treated as 1x1 matrices
  • Strings: 1xn matrices of characters

2.2 Basic Use of MATLAB

MATLAB is command line driven; all commands are entered after the prompt symbol (>>). It supports standard arithmetic operations and provides various output formats.

Basic Operations


>> 2+2    % Addition
ans = 4

>> 3*4    % Multiplication
ans = 12

>> 7-3    % Subtraction
ans = 4

>> 11/7   % Division
ans = 1.5714

>> 2^5    % Exponentiation
ans = 32

>> format long  % Change output format

Command Window Features

  • A blank window for entering commands
  • Prompt consists of two right arrows (>>)
  • Immediate feedback after each command

2.3 Variables

Variables in MATLAB store values and can be displayed in different formats while maintaining their full precision internally.

Variable Examples


a = 5^(7/2)           % Assigns value to variable a
b = sin(pi/9)-cos(pi/9)  % Assigns value to variable b
format long; a        % Displays full precision

2.3.1 The Workspace

Workspace Features

  • Lists all currently defined variables
  • Shows numeric data types
  • Displays sizes in bytes
  • Accessible via View menu or 'whos' command

Workspace Command


>> whos  % Displays information about variables in workspace

2.4 Matrices

Matrices are the fundamental data structure in MATLAB. Understanding how to work with matrices is essential for effective MATLAB programming.

2.4.1 Matrix Elements

Accessing Elements


>> a(2,3)      % Returns element in row 2, column 3
>> a(10)       % Single number indexing
>> 2:3:16      % Generates a vector of values
>> a(2, 1:3)   % Returns values in row 2, columns 1-3
>> a(2:4,3)    % Returns column 3, rows 2-4

Colon Operator


>> a(2:3,3:4)  % Lists 2x2 block from rows 2-3, cols 3-4
>> a(3, :)     % Gets all elements of row 3
>> a(:, 2)     % Gets all elements of column 2

2.4.2 Matrix Operations

Basic Operations


>> b=[2 4 -7 -4; 5 6 3 -2; 1 -8 -5 -3; 0 -6 7 -1]
>> 2*a-3*b     % Scalar multiplication and subtraction
>> a^3*b^4     % Matrix powers and multiplication
>> inv(a)      % Matrix inversion
>> a'          % Matrix transpose
>> flipud(a)   % Flip matrix up/down
>> fliplr(a)   % Flip matrix left/right
>> rot90(a)    % Rotate matrix by 90 degrees

Reshape and Dot Operations


>> c=[1:20]    % Create vector
>> reshape(c,4,5)  % Reshape into 4x5 matrix
>> a.*b        % Element-wise multiplication
>> a.^2        % Element-wise square
>> 1./a        % Element-wise reciprocal

Special Matrices

  • zeros(n) - Creates n×n matrix of zeros
  • ones(m,n) - Creates m×n matrix of ones
  • rand() - Random numbers in [0,1]
  • randn() - Normal distribution random numbers

2.5 Plots

MATLAB has outstanding graphics capabilities. You can create plots using vectors of x and y coordinates.

Basic Plotting


>> x=[0:0.1:2*pi];        % Create x vector
>> plot(x,sin(x))         % Plot sine function
>> plot(x,sin(x),'.',x,cos(x),'o')  % Plot multiple functions

Help in MATLAB

MATLAB comes with a vast amount of online help and information. To obtain information on a particular command, you can use the help function.
