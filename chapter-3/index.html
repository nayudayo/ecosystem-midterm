Chapter 3: Working with Digital Images in MATLAB

What is MATLAB?

MATLAB is a powerful computing environment used for numerical computing, data visualization, and algorithm development. It provides extensive support for matrix operations and image processing through its specialized toolboxes.

Key Features

  • Built-in functions for matrix operations
  • Comprehensive Image Processing Toolbox
  • Efficient handling of complex image processing tasks

Why Use MATLAB for Image Processing?

Advantages

  • Images can be manipulated as matrices
  • Large collection of built-in functions
  • Widely used in research and engineering
  • Excellent for image analysis and enhancement

Digital Images

A digital image consists of a grid of pixels, where each pixel represents a small portion of the image. Each pixel has a numerical value that defines its intensity or color.

Greyscale Images

Each pixel has a range of intensities from 0 (black) to 255 (white). These images contain only shades of gray and are stored as 2D matrices where each element represents a pixel's brightness.

Common Applications

  • Medical Imaging (X-rays, CT scans)
  • Document Scanning with OCR
  • Face Detection and Biometric Analysis

RGB Images

Each pixel consists of three color components: Red, Green, and Blue. RGB images are stored as three-dimensional arrays containing red, green, and blue channels.

Accessing Color Channels


redChannel = a(:, :, 1);   % Extract red
greenChannel = a(:, :, 2); % Extract green
blueChannel = a(:, :, 3);  % Extract blue

Indexed Colour Images

Uses a colormap where each pixel value points to a specific color in a predefined list. This format is efficient for storage and is particularly useful when working with a limited color palette.

Advantages

  • Reduced file size and data storage needs
  • Faster loading and rendering performance
  • Memory efficient for devices with limited resources

Loading and Displaying Images

Reading Images


>> w = imread('wombats.tif');  % Load image into variable w

Note: The semicolon prevents output display of the large matrix

Displaying Images


>> figure              % Create new window
>> imshow(w)          % Display the image
>> pixval on          % Enable pixel value display

Important Commands

  • imread: Reads image file into matrix
  • imshow: Displays image from matrix
  • figure: Creates new display window
  • pixval: Shows pixel values on hover

Getting Image Information

Use the imfinfo function to retrieve detailed information about an image file:


>> imfinfo('image.tif')    % Get detailed image information

Retrieved Information Includes:

  • Filename and FileModDate
  • FileSize in bytes
  • Width and Height in pixels
  • BitDepth and ColorType

Accessing Pixel Values

For RGB images, you can retrieve pixel values using impixel:


>> impixel(a,200,100)    % Get RGB values at column 200, row 100
>> a(100, 200, :)        % Alternative way to get RGB values

Note: For grayscale images, impixel returns three identical values since there's only one intensity value per pixel.

Image Processing Basics

MATLAB provides various functions for image processing tasks such as filtering, edge detection, and segmentation.

Common Operations

  • Image enhancement and restoration
  • Feature detection and extraction
  • Image transformation and analysis
  • Color space conversion

Data Types and Conversions

Understanding data types is crucial for efficient memory management and error-free image processing. MATLAB provides various functions for converting between different image types.

Image Type Conversion Functions


y = ind2gray(x,map);     % Indexed to grayscale
[y,map] = gray2ind(x);   % Grayscale to indexed
y = rgb2gray(x);         % RGB to grayscale
y = gray2rgb(x);         % Grayscale to RGB
[y,map] = rgb2ind(x);    % RGB to indexed
y = ind2rgb(x,map);      % Indexed to RGB

Common Data Types

  • uint8: Unsigned 8-bit integers (0-255)
  • double: Double-precision floating-point
  • logical: Binary values (0 or 1)
