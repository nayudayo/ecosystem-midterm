Point Processing

Chapter 5: Image Processing Operations

5.1 Introduction

Image processing operations transform pixel grey values through different approaches, ranging from simple to complex transformations.

Transforms

Requires knowledge of all grey levels in the entire image. The image is processed as a single large block.

Spatial Filters

Uses grey levels in a small neighborhood around each pixel for transformation.

Point Operations

Changes pixel grey values independently, without knowledge of surrounding pixels.

5.2 Arithmetic Operations

Arithmetic operations involve applying mathematical functions to pixel values in the range 0-255.

Basic Operations


y = x + C    % Adding a constant
y = Cx       % Multiplication by constant

Brightness Adjustment Rules

  • If result > 255, clamp to 255
  • If result < 0, clamp to 0
  • Otherwise, use calculated value

Brightness Effects

Adding Constants

  • Adding positive values lightens the image
  • Values ≥ (255 - constant) become 255
  • Reduces image contrast in bright regions

Subtracting Constants

  • Subtracting values darkens the image
  • Values ≤ constant become 0
  • Reduces image contrast in dark regions

5.3 Histograms

Histograms show the distribution of pixel intensities in an image, helping analyze and improve image quality.

Working with Images


>> b = imread('blocks.tif');    % Read image
>> whos b                       % Check image properties
>> imhist(b)                   % Display histogram

5.4 Thresholding

Thresholding is a point processing operation that converts grayscale images to binary images based on a threshold value.

Thresholding Process

  • Choose a threshold value T
  • Pixels > T become white (255)
  • Pixels ≤ T become black (0)
  • Results in binary image

5.5 Applications of Thresholding

Common Applications

  • Object segmentation
  • Text extraction
  • Document processing
  • Quality control inspection
